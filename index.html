Movie Finder

Open Page
This website is a project that I created for a submission in one of the Dicoding classes. It enables users to search for movies by entering the title or a part of it. It then displays a list of movies that match the search query, showing the full title, poster image, and a brief summary of each movie. It uses the OMDb API to fetch the movie data and display it on the web page. It is built with HTML, CSS, and JavaScript, and has a simple and user-friendly interface. It is a great way to find and explore movies.
